# Project Overview

## Purpose
Personal blog built with Next.js, currently serving as a portfolio site with MDX blog posts. The project is focused on AI and development content in Korean language, with plans to implement email subscription for post notifications.

## Current Features
- MDX and Markdown blog post support
- SEO optimized (sitemap, robots, JSON-LD schema)
- RSS Feed
- Dynamic OG images
- Syntax highlighting with sugar-high
- Dark/light theme support
- Vercel Analytics and Speed Insights
- Responsive design with Tailwind CSS

## Tech Stack
- **Frontend**: Next.js (canary), React 18.3.1, TypeScript 5.3.3
- **Styling**: Tailwind CSS 4.0.0-alpha.13
- **Content**: MDX with next-mdx-remote, marked for markdown processing
- **Analytics**: Vercel Analytics & Speed Insights
- **Package Manager**: pnpm
- **Deployment**: Vercel (inferred from analytics)
- **Testing**: Playwright for E2E testing
