# Task Completion Checklist

## Before Committing
1. **Build Check**: Run `pnpm build` to ensure no build errors
2. **Type Check**: Verify TypeScript types are correct
3. **Test**: Run `npx playwright test` for E2E tests
4. **Review**: Check all files follow project conventions

## Git Workflow
1. **Branch**: Create feature branch following naming convention
2. **Commit**: Use Conventional Commits format
3. **PR**: Use PR template in `.github/PULL_REQUEST_TEMPLATE.md`
4. **Reference**: Include relevant issue numbers

## Code Quality
1. **TypeScript**: Ensure proper typing throughout
2. **Components**: Follow established patterns
3. **Styling**: Use Tailwind consistently
4. **Accessibility**: Ensure WCAG compliance
5. **Performance**: Optimize for Core Web Vitals

## Deployment Considerations
1. **Vercel**: Project configured for Vercel deployment
2. **Environment**: Check environment variables are set
3. **Analytics**: Ensure tracking is working
4. **SEO**: Verify metadata and sitemap updates
